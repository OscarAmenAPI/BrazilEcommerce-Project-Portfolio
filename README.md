# Data-Analytics-Portfolio

Brazil E-Commerce Sales Analysis | SQL & Power BI

Project Summary:	
	This project analyzes a Brazilian e-commerce dataset to understand revenue performance, customer activity, product category performance, delivery efficiency, and customer satisfaction. The goal of the analysis is to identify which products and regions drive the most revenue, and how revenue changes over time. SQL was used to clean, join, and analyze the data, while Power BI was used to build an interactive dashboard displaying key business KPIs and trends.

Business Problem:
	The business needed a centralized dashboard to monitor sales, customer activity, regional performance, delivery time issues, and review scores across the marketplace.
